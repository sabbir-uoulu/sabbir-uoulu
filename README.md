### Hi there 👋

I'm a **Doctoral Researcher** at the [6G Flagship](https://www.6gflagship.com/), University of Oulu, Finland. My research is about AI-enabled waveform design and optimization for 6G wireless systems. However, as I have been working for several EU/national/international projects, I spent most of my time on the design and development of 5G/O-RAN end-to-end testbeds for different use cases. These days, testing and verifying **5G/O-RAN/6G/NTN** features using open-source tools and techniques is my strong passion. Previously at the **5G Test Network (5GTN) Oulu**, where I led Horizon Europe 6G-XR deliverable D1.1, Task T2.3 (AI-based slicing models), 5G!Drones' D3.3, and others, and worked on 5G/B5G testbed deployment with open-source and commercial RAN for UAV and XR use cases.

I build reproducible, end-to-end mobile network testbeds and care about doing the hard, honest engineering, down to the timing of a single random-access preamble.

### 🔬 Focus Areas

- 📡 **Open RAN & Core** — OpenAirInterface (OAI), srsRAN (O-CU/O-DU), Open5GS, free5GC, Magma Core, UERANSIM
- 🎛️ **O-RAN** — O-RAN SC, FlexRIC, near-RT/non-RT RIC, xApps/rApps
- 🛰️ **Non-Terrestrial Networks** — 5G NR NTN, GEO/LEO, direct-to-device (D2D), 3GPP Release 17/18/19
- 🤖 **AI/ML for Networks** — AI-enabled waveform design, network slicing models, RAN optimisation
- 🧪 **Testbed Engineering** — end-to-end 5G SA deployment, RF simulation, protocol validation
- ☁️ **Adjacent** — Docker/Kubernetes for CNFs, network automation, B5G/6G research

### 🚀 Featured Project

**[Open-Source 5G NR NTN Testbed](https://github.com/sabbir-uoulu/open-source-5g-ntn)** — a fully open-source testbed completing a complete **end-to-end GEO satellite attach** (sync → SIB19 → 4-step RACH → RRC → NAS registration → PDU session) on OpenAirInterface + Open5GS, including a protocol-level fix making the gNB random-access response window NTN-aware.
